üéØ Option 1: Create GitHub Repository (Recommended)

This enables automatic Vercel deployments:

  1. Create New GitHub Repo
  2. Repository name: CalyClub
  3. Make it Public
  4. Don't initialize with README (we have files already)
  5. Click "Create repository"

Then run these commands in your CalyClub folder:

git remote set-url origin https://github.com/YourUsername/CalyClub.git
git push -u origin master

Replace "YourUsername" with your actual GitHub username

üî• Option 2: Deploy to Vercel Dashboard

Once your GitHub repo is created:

  1. Deploy on Vercel
  2. Click "Import Git Repository"
  3. Select your CalyClub repository
  4. Configure project:
    • Project Name: calyclub
    • Framework: Other
    • Root Directory: ./ (default)
    • Build Command: (leave empty)
    • Output Directory: public
  5. Click "Deploy"
